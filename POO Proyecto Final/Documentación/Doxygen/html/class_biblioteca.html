Detailed Description

Clase contenedora que administra la Biblioteca. La misma será utilizada mediante un Singleton para evitar múltiples instancias.

Contiene la lista de libros, lectores, préstamos y sanciones, y los metodos para guardarlos y buscarlos en sus respectivas pestañas. Tambíen contiene los métodos para agregar, modificar y eliminar elementos. Los datos estan siempre en memoria, y al realizar alguna operación se debe llamar al metodo Guardar para persistir los cambios en los archivos.
